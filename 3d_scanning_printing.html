3D Printing and Scanning Assignment

By Enrique Groso

Overview

This project explores the fundamentals of additive manufacturing through two main components: designing and 3D printing an original object and scanning an existing object. Below, you'll find detailed documentation of the entire process.

Part 1: 3D Printed Design

Concept and Design

My design aimed to demonstrate the unique capabilities of additive manufacturing. The object is a geometric sculpture that features intricate details impossible to create with traditional subtractive methods.

3D Modeling

The design was created using Fusion 360 with the following specifications:

  • Size: 5x5x5 cm
  • Material: PLA
  • Layer Height: 0.2mm

Printing Process

The object was printed using a 3D Printer. Below are the settings used:

  • Print Speed: 60 mm/s
  • Infill: 20%
  • Support: None

Part 2: 3D Scanning

Object Selection

I chose a fidget spinner for its unique shape and features that would challenge the scanning process.

Scanning Process

The object was scanned using a Ferret Scanner. Key steps included:

  • Calibration of the scanner
  • Positioning of the object
  • Post-processing the scanned data

Reflection

This project taught me valuable lessons in design thinking and problem-solving. I learned how to leverage the unique strengths of additive manufacturing and how to overcome challenges in 3D scanning.
